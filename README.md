# Barbershop Website

Современный и адаптивный веб-сайт для барбершопа, созданный для привлечения новых клиентов и предоставления ключевой информации о услугах, мастерах и контактах.

## О проекте

Этот сайт представляет собой визитную карточку барбершопа в интернете. Он включает в себя:

*   **Главный экран:** Привлекательное представление с призывом к действию
*   **О нас:** Краткое описание философии и ценностей барбершопа
*   **Услуги и цены:** Детальный прайс-лист на все основные услуги
*   **Наша команда:** Блок с фотографиями и именами мастеров
*   **Галерея работ:** Раздел с фотографиями выполненных стрижек и стилей
*   **Контакты:** Адрес, телефон, электронная почта и интерактивная карта
*   **Форма обратной связи:** Позволяет клиентам легко связаться с барбершопом

## Технологии

Проект реализован с использованием следующих технологий:

*   **HTML5** — семантическая разметка
*   **CSS3** — стилизация, адаптивная верстка и анимации
*   **JavaScript** — интерактивность и динамическое поведение

## Установка и Запуск

Чтобы запустить проект локально на своем компьютере, выполните следующие шаги:

### Инструкция по запуску

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/BedrinDG/barbershop-website.git
    ```
2.  **Перейдите в директорию проекта:**
    ```bash
    cd barbershop-website
    ```
3.  **Установите зависимости**

    ```bash
    npm install
    ```
        
4.  **Готово! Можно запускать файл index.html**
